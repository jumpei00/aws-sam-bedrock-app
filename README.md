## AWS Sam And Bedrock Application
### Preparation
- pyenv install

```bash
brew install pyenv
```

- python ^3.11 install

```bash
pyenv install 3.11.3
pyenv global 3.11.3
```

- poetry install

```bash
curl -sSL https://install.python-poetry.org | python3 -
echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.zshrc
```
